# SV Attributes

Reusable Webflow attributes and utilities by Samvrit Studio.

## Installation

Add the script before the closing `</body>` tag:

```html
<script
  type="module"
  src="https://cdn.jsdelivr.net/gh/samvrit-studio/sv-attributes/dist/sv-attributes.js"
></script>
```

## Attributes

### sv-current-year

Automatically inserts the current year.

#### HTML

```html
<span sv-current-year></span>
```

#### Output

```text
2026
```
